Math Object Methods and Properties

Math.PI: Represents the mathematical constant π (pi), approximately 3.14159.
Example: console.log(Math.PI); // Output: 3.141592653589793
Math.abs(x): Returns the absolute value of a number x.
Example: console.log(Math.abs(-5)); // Output: 5
Math.ceil(x): Returns the smallest integer greater than or equal to a number x.
Example: console.log(Math.ceil(4.3)); // Output: 5
Math.floor(x): Returns the largest integer less than or equal to a number x.
Example: console.log(Math.floor(4.9)); // Output: 4
Math.round(x): Rounds a number x to the nearest integer.
Example: console.log(Math.round(4.5)); // Output: 5
Math.max(x1, x2, ..., xn): Returns the largest of zero or more numbers x1, x2, ..., xn.
Example: console.log(Math.max(10, 20, 30)); // Output: 30
Math.min(x1, x2, ..., xn): Returns the smallest of zero or more numbers x1, x2, ..., xn.
Example: console.log(Math.min(10, 20, 30)); // Output: 10
Math.random(): Returns a pseudo-random number between 0 (inclusive) and 1 (exclusive).
Example: console.log(Math.random()); // Output: A random number between 0 and 1 (exclusive)
Math.sqrt(x): Returns the square root of a non-negative number x.
Example: console.log(Math.sqrt(25)); // Output: 5
Math.pow(x, y): Returns the base x raised to the power y.
Example: console.log(Math.pow(2, 3)); // Output: 8 (2 raised to the power of 3)
Math.sin(x), Math.cos(x), Math.tan(x): Returns the sine, cosine, and tangent of an angle x (in radians), respectively.
Example: console.log(Math.sin(Math.PI / 2)); // Output: 1 (sin(π/2) = 1)
Math.asin(x), Math.acos(x), Math.atan(x): Returns the inverse sine, cosine, and tangent of a number x, respectively (result is in radians).
Example: console.log(Math.asin(1)); // Output: 1.5707963267948966 (inverse sin of 1 is π/2)
